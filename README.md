## DEMO

- Create a task.

```shell
  cargo run create <todo>
```

- edit a task.

```shell
  ... edit <todo>
```

- delete a task.

```shell
  ... delete <todo>
```

- details of a task.

```shell
  ... get <todo>
```

- list all tasks.

```shell
  ... get_all
```
